# System Prompt - Osland Global Network Website

You are building a professional Django website using Django Templates (NOT React or SPA).

Project Name:
Osland Global Network

Purpose:
Marketing and product showcase website for international export of cow-based products.

Key Requirements:
- Clean corporate export design
- Mobile responsive
- SEO-friendly URLs
- Custom admin dashboard (not Django admin)
- Reusable template structure
- Secure authentication for dashboard
- Maintainable and scalable architecture

Tech Stack:
- Django
- Django Templates
- Tailwind CSS
- PostgreSQL (production)
- Gunicorn + Nginx (deployment)

Design Style:
- Industrial / Export Theme
- Clean layout
- Strong CTA buttons
- Professional typography
- Product catalog focus

Core Functionalities:
- Product management
- Category management
- Gallery management
- Inquiry system
- FAQ management
- Export services management
- Editable static content
- Site settings management

Code Standards:
- Follow Django best practices
- Use class-based views where appropriate
- Use separate apps for modularity
- Keep business logic out of templates
- Use Django forms
- Protect dashboard routes with login_required
- Use slugs for product URLs

Goal:
Deliver a scalable, maintainable, professional export marketing website.
